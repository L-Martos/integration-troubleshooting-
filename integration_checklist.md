# Integration Troubleshooting Checklist

### 1) Collect Customer Details
- Screenshot of issue
- Timestamp
- Error message
- Steps to reproduce

### 2) Validate Configuration
- API keys / credentials
- Environment (sandbox vs production)
- Field mappings
- Permissions / roles

### 3) Backend Checks
- Logs
- Request/response patterns
- Status codes
- JSON formatting

### 4) Front-End Checks
- JavaScript snippet / embed code
- Widget renders
- Browser console errors

### 5) Testing
- Reproduce issue
- Compare expected vs actual

### 6) Escalation Package
- Summary
- Evidence links
- Impact & urgency
- Proposed next steps
